About This Project

This website was created as a school project for the Data Structures and Algorithms course at Wuhan University, Year 1 Semester 2. Created by Alinane Kamanga, Khongkwan Leksomboon and Rachel Felicia.

The project aims to provide a more visual and interactive learning experience through graph visualization, search systems, algorithm explanations, code examples, complexity analysis, and educational diagrams.

This project is intended strictly for educational and non-commercial purposes.

Some diagrams and visual references are adapted from online educational resources for learning and classroom presentation use.
